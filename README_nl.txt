Een simple app om je Wallbox laadpaal te bedienen via de WallBox API.

Start en stop het laden van je auto op een tijdschema of basseer het op condities in een flow. Voer Homey flow uit wanneer je auto start met laden of stuur een notificatie om je eraan te herinner om je auto op teladen. Of zet je laadpaal uit wanneer je voor langere tijd weg bent.
